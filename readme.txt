Easily add custom scripts like JavaScript, CSS, or HTML to your WordPress site's header and footer without editing theme files. Includes enable/disable toggle and placeholder guidance for non-technical users. Fully translation-ready and secure.

== Description ==

WP Header & Footer Injector is a lightweight plugin to safely add custom code to your WordPress site.  
- Inject scripts in `<head>` or before `</body>`  
- Enable or disable injection with a single toggle  
- Placeholders guide users for JavaScript and CSS  
- Works with any theme without editing theme files  
- Translation-ready for international use  
- Sanitized inputs for security  

Perfect for Google Analytics, Facebook Pixel, custom CSS, or other tracking scripts.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to the **Header & Footer** menu in the sidebar
4. Add your code in header or footer and enable/disable injection

== Frequently Asked Questions ==

= Can I use this plugin with any theme? =

Yes, it works with all WordPress themes without modifying theme files.

= Is it safe to add JavaScript code? =

Yes, all inputs are sanitized and validated. Only allowed HTML, JS, CSS tags are processed.

= Can I disable code injection temporarily? =

Yes, use the enable/disable toggle on the settings page.
